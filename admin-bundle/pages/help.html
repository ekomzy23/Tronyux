Frequently Asked Questions

How do I create an exam? –¼
Go to Exams from the bottom navigation, then tap the ＋ button in the top-right corner. Fill in the exam title, course code, year level, duration, and question type. Once ready, tap Publish to make it visible to students.
How do I publish results? –¼
Go to Results from the More menu. Select the exam you want to publish results for. Review all submitted attempts, then tap Publish Results. Students will be notified automatically and can view their scores in the student app.
How do I add students? –¼
Go to Students in the bottom navigation. Tap the ＋ FAB button at the bottom right to add a single student manually. For bulk imports, go to More → Bulk Import and upload a CSV file with student data.
How do I use the AI Center? –¼
Go to More → AI Center. You can generate questions by entering a topic, selecting the question type and difficulty. Use the Question Tools to detect duplicates, analyse difficulty, or improve existing questions. The AI Console lets you send free-form prompts. Note: full AI features require an API key configuration.
How do I import questions in bulk? –¼
Go to More → Bulk Import. Select Questions as the import type. Download the CSV template shown in the Format Guide, fill in your questions following the column format, then upload the file. The system will validate and import all valid rows.
What question types are supported? –¼
Ronyx supports MCQ (Multiple Choice with 4 options), Theory (long-form essay answer), True/False, and Fill in the Blank. Theory questions require manual marking from the admin panel via Theory Marking.
How does the Live Monitor work? –¼
Go to More → Live Monitor during an active exam. Select the exam from the chips at the top to filter candidates. You can see each student's progress (questions answered), time remaining, and flag status. The list auto-refreshes every 15 seconds.
How do I set up Appwrite? –¼
Edit the file js/config.js and fill in your Appwrite Endpoint, Project ID, Database ID, and Collection IDs. Set CONFIGURED: true and reload. All admin features will activate automatically once connected.
